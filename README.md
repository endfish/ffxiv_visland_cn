# ffxiv_visland_cn

A plugin to automate nearly everything involving Island Sanctuary.

Repo
```
https://puni.sh/api/repository/veyn
```

For presets, see the Wiki: https://github.com/awgil/ffxiv_visland/wiki

## 中文说明

这是 `awgil/ffxiv_visland` 的中文化分支，当前主要包含：

- 插件发布名为 `Visland-CN`
- 插件命令为 `/vislandcn`
- 简体中文界面翻译
- 国服 `XIVLauncherCN` / Dalamud 开发环境适配
- 工坊排班剪贴板导入适配国服腾讯文档格式

### 国服工坊排班导入

当前 `OC 导入` 同时支持：

- 原版 Overseas Casuals / Discord 英文排班格式
- 国服腾讯文档常见格式，例如：

```text
D1:休息
D2:3×新薯沙拉、五海杂烩汤、无人面包、五海杂烩汤
D3:3×墨鱼汁、鲨油、墨鱼汁、鲨油
D4:休息
D5:3×玉米片、腌小萝卜、玉米片、腌小萝卜
D6:3×水牛豆沙拉、角笛、绵羊地毯、床
D7:3×黄油、绵羊地毯、床、骑士帽
```
